# API EARTHQUAKE


## Instalacion

```bash
$ bundle install
```
**Pasos para correr aplicacion**

## Running 

```bash
$ rails db:migrate
$ rails db:schema:load
$ rails s
```

## Task
**Correr task para obtener datos desd el sitio (earthquake.usgs.gov)**

```bash
$ rails earthquake:get_data
```
